## Setup

#### Install deps:

```bash
yarn
```

#### Start `cli` :computer:

```bash
yarn start
```

#### Run the app :iphone:

Android :robot: :
```bash
yarn android
```

> or
iOS :apple: :
```bash
yarn ios
```

#### Run the tests

```bash
yarn test
```
